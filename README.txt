Instrucciones de compilacion
1. Abrir terminal de linux en la carpeta
2. Ejecutar comando g++ tictactoe.cpp -o ./tictactoe
3. Ejecutar comando ./tictactoe
4. Ingresar inputs por consola (OPCIONAL: Hay un codigo con pythongui que pone el input solo, para cambiar el ejemplo solo cambiar la lista)
5. Ejecutar comando g++ loteria.cpp -o ./loteria
6. Ejecutar comando ./loteria
7. incribir input por pantalla
